Key design decisions

  • Hexagonal boundary: include/ never imports Airflow. DAGs are orchestration only.
  • Streaming > lists: CsvSource returns iterators. Bounded memory regardless of input size.
  • Idempotency at the sink: every SqliteSink.write() uses INSERT OR IGNORE on a natural key, so retries are safe.
  • Static analysis for the UI: the web app reads DAG files via AST. It works without Airflow installed and never side-effects the DAG bag.
  • TAOCP-style records: every record is a frozen=True, slots=True, kw_only=True dataclass with explicit NewType ids.

Full reasoning: docs/design_decisions.md, docs/patterns.md, docs/taocp_applications.md.
